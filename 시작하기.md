# 🚀 산그린호텔 웹사이트 - 빠른 시작 가이드

## 1️⃣ 설치하기

먼저 필요한 패키지들을 설치합니다:

```bash
npm install
```

또는 pnpm을 사용하는 경우:

```bash
pnpm install
```

## 2️⃣ 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173`을 열어 확인하세요!

## 3️⃣ 프로덕션 빌드

웹사이트를 배포하기 위해 빌드합니다:

```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

## 4️⃣ 빌드 미리보기

빌드된 결과물을 로컬에서 확인:

```bash
npm run preview
```

## 📁 주요 파일 설명

- `src/app/App.tsx` - 메인 애플리케이션
- `src/imports/0메인디자인.tsx` - Figma 디자인 컴포넌트
- `src/styles/` - 스타일 파일들
- `index.html` - HTML 진입점
- `package.json` - 프로젝트 설정 및 의존성

## 🌐 배포하기

자세한 배포 방법은 `DEPLOYMENT_KR.md` 파일을 참고하세요!

가장 쉬운 방법은:
1. GitHub에 코드 업로드
2. [Vercel](https://vercel.com)에서 배포

## 💡 도움말

- **포트 변경**: `vite.config.ts`에서 설정
- **빌드 오류**: `node_modules` 삭제 후 재설치
- **폰트 문제**: `src/styles/fonts.css` 확인
