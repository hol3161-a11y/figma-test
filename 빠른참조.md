# 🚀 빠른 참조 카드

## 자주 사용하는 명령어

```bash
# 개발 시작
npm run dev

# 빌드
npm run build

# 미리보기
npm run preview

# 패키지 재설치
rm -rf node_modules && npm install
```

## 프로젝트 주소

- **개발 서버**: http://localhost:5173
- **배포 URL**: (배포 후 기입)

## 주요 파일 위치

| 파일 | 경로 | 용도 |
|------|------|------|
| 메인 앱 | `/src/app/App.tsx` | 애플리케이션 진입점 |
| 디자인 | `/src/imports/0메인디자인.tsx` | Figma 디자인 |
| 폰트 | `/src/styles/fonts.css` | 폰트 설정 |
| 테마 | `/src/styles/theme.css` | 색상/변수 |
| HTML | `/index.html` | HTML 템플릿 |

## 빠른 배포 가이드

### Vercel (가장 빠름)
1. GitHub에 push
2. vercel.com 접속
3. Import Project
4. 완료!

### Netlify
1. GitHub에 push
2. netlify.com 접속
3. Add new site
4. 완료!

## 문제 해결

| 문제 | 해결 방법 |
|------|----------|
| 빌드 실패 | `rm -rf node_modules && npm install` |
| 포트 이미 사용 중 | 5173 포트 종료 또는 다른 포트 사용 |
| 이미지 안 보임 | `public` 폴더 확인 |
| 404 에러 | `.htaccess` 또는 설정 파일 확인 |
